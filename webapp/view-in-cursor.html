To see the app inside Cursor:

1. Make sure the server is running (Terminal: cd webapp && python -m http.server 5500)
2. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
3. Type Simple Browser
4. Choose Simple Browser: Show
5. Enter: http://localhost:5500

Or click to open in system browser: http://localhost:5500
